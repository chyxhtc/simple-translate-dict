# Simple Translate Dict

这个项目是从原Simple Translate项目拉取的源代码，在此基础上增加了字典查询和发音的功能。

目前该项目仅用于测试，尚未发布正式版本。

## 新增功能
- 字典查询功能
- 单词发音功能

## 原项目
本项目基于Simple Translate开发，保留了原有的翻译功能。

https://github.com/sienori/simple-translate
